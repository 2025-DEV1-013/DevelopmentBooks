# Development Books Backend

This project is a Spring Boot 4.0.0 backend service built using Java 21.  
It serves as the foundation for implementing the DevelopmentBooks Kata using a clean architecture and TDD-driven approach.

---

## Project Setup

The initial setup includes:

- **Spring Boot 4.0.0** project structure
- **Java 21** compatibility
